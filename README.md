# SoftwareDevOps_BuddyTracker
Project Name – Buddy Ticket Tracker (SEDO)

Overview – Shows “ready for buddy” tickets, shows ageing, nudges follow-ups.

Key Features – Login & roles. Ticket lifecycle. Ageing indicators (green/amber/red). Audit trail

Data Model (4 tables) – users, roles, user_roles, tickets

Roles & Permissions – Manager (Admin delete/force-close) Tester/Dev/BA (create/edit/view) Guest (read-only)

Testing Strategy - (BDD)

DevOps & CI/CD – GitHub Actions

Local Run - Flask steps

Deployment – Render
